Local Image — pick an image file from your device to use as the avatar instead of fetching online.
Reset — clears all rankings and reloads the full default list with fresh avatars.
Export JSON — saves your current rankings to a .json file you can back up or share.
Import JSON — restores rankings from a previously exported .json file.
Export Image — captures the tier rows as a PNG you can save or share.
